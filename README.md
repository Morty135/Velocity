# Velocity (Raylib)

- custom physics and collision resolution
- input abstraction
- animation handling
- scalable hitboxes decoupled from sprites

---

## Features

- Player physics
- Ground and wall detection
- Wall jumping
- Sprite-based animations

---

## Controls

| Action        | Keyboard | controller |
|--------------|-------|--------|
| Move Left    | A / Left | LeftJoystic |
| Move Right   | D / Right | LeftJoystic |
| Jump         | Space | A(Xbox) X(PS) |
| Wall Jump    | Jump+Move towards wall |
